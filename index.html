An essentially four-dimensional counterexample to the Jacobian conjecture

Arnab Maiti · 20 July 2026 · found and verified with Claude (Fable 5) · code & verification scripts

TL;DR. Following Levent Alpöge's July 19 counterexample to the Jacobian conjecture in \(\mathbb{C}^3\), we run the underlying cubic-factorization construction one degree up and obtain an explicit degree-10 polynomial map \(F:\mathbb{C}^4\to\mathbb{C}^4\) with integer coefficients, constant Jacobian determinant \(-2\), and a verified seven-point fiber. Its generic fiber has at least 7 points, so it is not a re-coordinatization or identity-padding of any known three-dimensional example (which have generic fiber sizes 3, 4, …, 100, or powers of 3): it is, to our knowledge, the first published counterexample that is genuinely four-dimensional. All claims are certified by exact rational arithmetic in SymPy; run the scripts yourself.

1. Background

The Jacobian conjecture (Kraus 1884 in two variables; Keller 1939 in general) asserted that every polynomial map \(\mathbb{C}^n\to\mathbb{C}^n\) whose Jacobian determinant is a nonzero constant has a polynomial inverse. On 19 July 2026 Levent Alpöge announced an explicit counterexample in \(\mathbb{C}^3\), found with the AI model Claude Fable 5: a map with \(\det JF\equiv-2\) sending three distinct points to one point. A reproducible derivation of that map (the cubic-factor and resultant construction) reveals the mechanism: parametrize factorizations of a cubic \(L(t)\,Q(t)\) with \(L\) linear and \(Q\) quadratic, normalize the resultant \(\rho=\operatorname{Res}(L,Q)\) to a constant to make the coefficient map étale, and observe that a cubic has three choices of linear factor, so the map is generically 3-to-1. Within a day, jacobianfun.org published a family realizing every generic fiber degree \(n\ge 3\), explicitly staying inside \(\mathbb{C}^3\).

This note asks the orthogonal question: does the factorization mechanism itself generalize upward in dimension? It does.

2. The map

Write coordinates \((x, y_1, y_2, y_3)\) on \(\mathbb{C}^4\). Define \(F = (F_1, F_2, F_3, F_4)\):

$$\begin{aligned} F_1 ={}& x^3y_2 - 2x^2y_1 + x,\\[4pt] F_2 ={}& -2x^4y_1^3y_2 + 4x^3y_1^4 - 7x^3y_1^2y_2 + 12x^2y_1^3 - 8x^2y_1y_2 + x^2y_3\\ & + 10xy_1^2 - 3xy_2 + y_1,\\[4pt] F_3 ={}& -4x^4y_1^4y_2 + 8x^3y_1^5 - 14x^3y_1^3y_2 + 24x^2y_1^4 - 18x^2y_1^2y_2 + 2x^2y_1y_3\\ & + 24xy_1^3 - 10xy_1y_2 + 2xy_3 + 7y_1^2 - 2y_2,\\[4pt] F_4 ={}& -2x^4y_1^5y_2 + 4x^3y_1^6 - 8x^3y_1^4y_2 + 14x^2y_1^5 - 12x^2y_1^3y_2 + x^2y_1^2y_3\\ & + 17xy_1^4 - 8xy_1^2y_2 + 2xy_1y_3 + 7y_1^3 - 2y_1y_2 + y_3. \end{aligned}$$

Theorem. \(\det JF = -2\) identically, yet \(F\) is not injective: the fiber over \((0,0,-1,0)\) contains the seven distinct points below, including the three-point rational collision

$$F(1,-1,-3,-1) \;=\; F(-1,1,-3,1) \;=\; F\!\left(0,0,\tfrac12,0\right) \;=\; (0,0,-1,0).$$

Both statements are finite computations in exact rational arithmetic (no floating point, no genericity assumptions), certified by the scripts below. In particular the Jacobian conjecture fails in \(\mathbb{C}^4\) by an example native to that dimension.

The full seven-point fiber, corresponding to the target quartic \(t^3 - t = t(t-1)(t+1)\):

PointSheet (choice of linear factor)
\((1,-1,-3,-1)\)\(L \sim t\)
\((-1,1,-3,1)\)\(L \sim -t\)

\(\left(\tfrac{i\sqrt2}{2},\,-1+i\sqrt2,\,6+2i\sqrt2,\,-2-i\sqrt2\right)\) and conjugate\(L \sim t-1\)

3. The construction

Take \(L(t)=xt+\beta\) and a cubic \(Q(t)=\gamma_3t^3+\gamma_2t^2+\gamma_1t+\gamma_0\), so \(L\cdot Q\) is a quartic with coefficients \(c_4,\dots,c_0\), and let \(\rho=\operatorname{Res}(L,Q)\). The coefficient–resultant map satisfies the exact polynomial identity

$$\det\frac{\partial(c_4,c_3,c_2,c_1,c_0,\rho)}{\partial(x,\beta,\gamma_3,\gamma_2,\gamma_1,\gamma_0)} = -2\rho^2,$$

so on the slice \(Y=\{c_3=1,\ \rho=-1\}\subset\mathbb{A}^6\) the retained coefficient map \((c_4,c_2,c_1,c_0)\) is everywhere étale. (The value \(\rho=-1\) is the rational branch: \(\rho=+1\) forces \(\beta^2\equiv-1\bmod x\).) A polynomial chart of \(Y\) is found exactly as in the cubic case, cancelling obstructions order by order in \(x\):

$$\beta = 1+xy_1,\qquad \gamma_3 = 1-2xy_1+x^2y_2,\qquad \gamma_2 = \frac{1-\beta\gamma_3}{x},\qquad \gamma_1 = w_1+xy_3,\qquad \gamma_0 = y_1w_1+\beta y_3,$$

with \(w_1 = \bigl(2\beta^2\gamma_2-y_1(\beta+1)\bigr)/x\); every apparent division cancels exactly. Substituting the chart into \((c_4,c_2,c_1,c_0)\) yields the map above.

One structural caveat, and why it is harmless: unlike the cubic case, the chart is not an isomorphism onto \(Y\). The rescaling \((L,Q)\mapsto(-L,-Q)\) acts freely on \(Y\) while fixing every \(c_i\) and \(\rho\), and Smith theory forbids a free \(\mathbb{Z}/2\) action on contractible affine space, so \(Y\not\cong\mathbb{A}^4\). The chart is an injective polynomial immersion with dense image; since \(\det JF=-2\) and the seven collisions are verified directly on the final polynomials, nothing in the theorem depends on the chart being surjective. The same obstruction proves that the analogous direct quartic construction inside \(\mathbb{C}^3\) is impossible, which is why "one degree up" must also mean "one dimension up."

4. The general pattern

For \(L\) linear times \(Q\) of degree \(m\) (total degree \(n=m+1\), in dimension \(n\)), the identity generalizes, verified symbolically for \(m=2,3,4\):

$$\det\frac{\partial(c_n,\dots,c_0,\rho)}{\partial(\text{factor coefficients})} = \pm(m-1)\,\rho^2 .$$

So the construction produces counterexamples native to every dimension \(n\ge3\) in characteristic zero. Because fixing \(\rho\) leaves a residual \(\mu_{m-1}\) scaling ambiguity per root, the generic fiber size is \(n(n-2)\): size 3 in \(\mathbb{C}^3\) (Alpöge's map), 8 for the map above, 15 in \(\mathbb{C}^5\), …

Since generic fiber cardinality is invariant under pre- and post-composition with polynomial automorphisms, and every previously published example has generic fiber size in \(\{3,4,5,\dots,100\}\cap\mathbb{C}^3\)-land or \(3^k\) (compositions) with padding preserving those values, the certified 7-point fiber (generic degree \(\ge 7\) by Zariski's main theorem) separates this map from all of them.

5. Verify it yourself

All scripts use exact rational arithmetic. With uv installed:

git clone https://github.com/arnabmaiti-gif/jacobian-c4.git
cd jacobian-c4/scripts
uv run --with sympy python independent_check_dim4.py  # det = -2 + rational 3-collision, from the polynomials alone
uv run --with sympy python verify_all.py              # full construction, all 7 fiber points, identity table

independent_check_dim4.py is deliberately minimal: it takes only the four displayed polynomials (not the construction) and certifies the theorem in a few seconds. construct.py and fiber_direct.py rebuild the map from the factorization construction and derive the fiber.

6. Honest caveats

  • The theorem (constant determinant + non-injectivity) is machine-certified. Two secondary claims currently rest on argument rather than certificate: that the displayed fiber has exactly 7 points (factorization classification; the direct Gröbner run was too slow), and the generic count \(n(n-2)=8\) (Zariski's main theorem gives \(\ge 7\) unconditionally from the certified fiber).
  • Nothing here is peer-reviewed, and the field is ~36 hours old. We searched the public record (arXiv, jacobianfun and its repo, secondary coverage) on 20 July 2026 and found no prior four-dimensional construction beyond identity-padding, but X/Twitter is not fully searchable and independent discovery is likely: the construction is the natural next step from the public derivation of the \(\mathbb{C}^3\) map.
  • The two-variable Jacobian conjecture remains open, as does the Dixmier conjecture for the first Weyl algebra.
